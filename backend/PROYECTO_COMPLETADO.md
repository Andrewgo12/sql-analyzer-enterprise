## 🚀 CÓMO USAR EL PROYECTO

### Inicio Rápido
```bash

# Opción 2: Manual

# Terminal 2 - Frontend
cd frontend

npm run dev
```

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
